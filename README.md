# 🎮 TETRIS NÉON // 2084

Un **Tetris futuriste** au style cyberpunk et néon, avec des effets visuels modernes (glow, particules, sons synthétiques).

Jouable directement dans le navigateur, parfaitement adapté à l'iPhone et aux écrans tactiles.

## ✨ Fonctionnalités

- Thème néon cyberpunk (cyan, magenta, violet)
- Effets de lueur (glow) sur les blocs
- Système de particules lors de la complétion de lignes
- Sons rétro-futuristes (Web Audio API)
- Preview de la pièce suivante
- Augmentation progressive de la vitesse
- Score, niveau, lignes
- Contrôles clavier + boutons tactiles optimisés mobile
- Design responsive et immersif

## 🚀 Déploiement sur Vercel (recommandé)

1. Va sur [https://vercel.com/new](https://vercel.com/new)
2. Connecte-toi avec GitHub
3. Importe le repo `tetris-futuriste`
4. Clique sur **Deploy** (c'est gratuit et ultra rapide)

Une fois déployé, tu auras une URL du style `https://tetris-futuriste.vercel.app`

## 🎭 Comment jouer

### Clavier (Desktop)
- **Flèches** ou **WASD** : Déplacer la pièce
- **↑** ou **X** : Tourner la pièce
- **Espace** : Chute rapide (hard drop)
- **P** : Mettre en pause
- **R** : Recommencer

### Mobile / iPhone (tactile)
Utilise les gros boutons en bas de l'écran :
- **⬅️** : Gauche
- **⟳** : Tourner
- **➡️** : Droite
- **⬇️** : Descendre doucement
- **⏬** : Chute instantanée

## 💻 Structure du projet

- `index.html` : Page principale du jeu

- `script.js` : Logique complète du Tetris + effets
